# mk-ca-bundle
Make ca-bundle.pem for broken FTPS servers.

## Usage

```console
$ git clone https://github.com/siglite/mk-ca-bundle
$ cd mk-ca-bundle
$ ./mk-ca-bundle > /path/to/ca-bundle.pem
```

## Broken FTPS servers

- \*.xrea.com (missing intermediate certificate)
